# src/neuralnetwork

This package contains the code scripts for network training, network evaluation, and GUI usage.

## Contents

- `training.py`: Core math, MNIST download, network initialization and training.
- `evaluation.py`: Accuracy and cost evaluation.
- `gui.py`: PyQt6 interface for drawing digits and training controls.
- `paths.py`: Central path definitions.
- `assets/`: Files needed by GUI.
