Neural Network for Handwritten Digits

Welcome to my little AI project, where I've created and trained a neural network from scratch only by math and basic programming.
This network is capable of detecting handwritten numbers between 0-9 by either an already trained network or YOUR own network.

The pretrained network has been trained by the MNIST dataset for about 60 hours. The MNIST dataset consists of 60.000 handwritten digits within a 28x28 pixel canvas. Above you can see its layout. It has 784 input neurons, 1 for each pixel of the 28x28 canvas
